\# 爱心弹窗生成器 💖



一个基于Web的爱心弹窗生成器，可以将温馨的提示语以爱心形状显示在屏幕上。



\## 功能特点



\- 🎨 美丽的爱心形状弹窗布局

\- 💌 随机显示温馨提示语

\- ⚙️ 可自定义弹窗数量和生成速度

\- 📱 响应式设计，支持各种设备

\- 🖱️ 点击弹窗即可关闭



\## 使用方法



1\. 调整"弹窗数量"滑块控制爱心的大小和密度

2\. 调整"生成速度"滑块控制弹窗出现的快慢

3\. 点击"生成爱心弹窗"按钮开始

4\. 点击任意弹窗可将其关闭

5\. 点击"清除所有弹窗"一键清除所有弹窗



\## 技术栈



\- HTML5

\- CSS3 (渐变、动画、阴影效果)

\- JavaScript (Canvas绘图、DOM操作)
